# cozyshelter-bgm

Background music assets for the **Cozy Shelter** blog ([cozyshelter.tistory.com](https://cozyshelter.tistory.com)).
Served via the [jsDelivr](https://www.jsdelivr.com/) CDN.

## Usage

Files are served from the repository root through jsDelivr:

```
https://cdn.jsdelivr.net/gh/MerciHanrim/cozyshelter-bgm@main/<filename>.mp3
```
